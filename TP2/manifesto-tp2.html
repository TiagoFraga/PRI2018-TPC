RESUMO:

O XML gerado a partir do DTD (Document Type Definition) juntamente com o potente processamento via XSLT fornece enormes facilidades no campo do tratamento da informação, padronizando métodos do mesmo num formato standard auxiliando assim o crescimento desta tecnologia no meio empresarial.

Este trabalho é composto por um DTD sobre a página web (http://clav.dglab.gov.pt/classes/consultar/c150.20.404) sobre um processo de negócio, uma stylesheet, uma página html que é gerada a partir da aplicação do referido stylesheet ao ficheiro xml que tambem está contido no conjunto de resultados pretendidos.
